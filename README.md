# Autopilot Launch Kit

A small marketing + intake site for the Lead-to-Booking Autopilot.

## What it includes (v0)
- Landing page (benefits, pricing, FAQ)
- Intake form (demo-first)
- Book-a-call CTA (Calendly link placeholder)
- Demo link to Lead-to-Booking Autopilot
- Vercel-ready Next.js setup

## Tech
- Next.js (App Router) + TypeScript
- Tailwind CSS

## Quickstart
1) npm install
2) npm run dev

## Deploy (Vercel)
1) Import this repo into Vercel
2) Framework auto-detect: Next.js
3) Deploy

## Privacy
Demo-first. No personal data required. Integrations can be added later.
